## Installation
* Install NPM.
* Clone the repository.
```
  $ git clone https://github.com/the-bose/stonks
```
* 'cd' into 'stonks' directory.
```
  $ cd stonks
```
* Install dependencies.
```
  $ npm install
```
* Start the program
```
  $ npm start
```
